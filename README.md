## Overview

### The challenge

Requirements
- Build a password generator that generates two random passwords when the user clicks the button.
- Each password should be 15 characters long

Stretch goals
- Ability to set password length
- Add "copy-on-click"
- Toggle "symbols" and "numbers" on/off
